Spring Boot Starters

2017-10-15

Spring Boot StarterжҳҜеңЁSpringBootз»„д»¶дёӯиў«жҸҗеҮәжқҘзҡ„дёҖз§ҚжҰӮеҝөпјҢstackoverflowдёҠйқўе·Із»ҸжңүдәәжҰӮжӢ¬дәҶиҝҷдёӘstarterжҳҜд»Җд№ҲдёңиҘҝпјҢжғізңӢе®Ңж•ҙзҡ„еӣһзӯ”жҲіиҝҷйҮҢ

Starter POMs are a set of convenient dependency descriptors that you can include in your application. You get a one-stop-shop for all the Spring and related technology that you need, without having to hunt through sample code and copy paste loads of dependency descriptors. For example, if you want to get started using Spring and JPA for database access, just include the spring-boot-starter-data-jpa dependency in your project, and you are good to go.

еӨ§жҰӮж„ҸжҖқе°ұжҳҜиҜҙstarterжҳҜдёҖз§ҚеҜ№дҫқиө–зҡ„synthesizeпјҲеҗҲжҲҗпјүпјҢиҝҷжҳҜд»Җд№Ҳж„ҸжҖқе‘ўпјҹжҲ‘еҸҜд»ҘдёҫдёӘдҫӢеӯҗжқҘиҜҙжҳҺгҖӮ

дј з»ҹзҡ„еҒҡжі•

еңЁжІЎжңүstarterд№ӢеүҚпјҢеҒҮеҰӮжҲ‘жғіиҰҒеңЁSpringдёӯдҪҝз”ЁjpaпјҢйӮЈжҲ‘еҸҜиғҪйңҖиҰҒеҒҡд»ҘдёӢж“ҚдҪңпјҡ

  1. еңЁMavenдёӯеј•е…ҘдҪҝз”Ёзҡ„ж•°жҚ®еә“зҡ„дҫқиө–пјҲеҚіJDBCзҡ„jarпјү
  2. еј•е…Ҙjpaзҡ„дҫқиө–
  3. еңЁxxx.xmlдёӯй…ҚзҪ®дёҖдәӣеұһжҖ§дҝЎжҒҜ
  4. еҸҚеӨҚзҡ„и°ғиҜ•зӣҙеҲ°еҸҜд»ҘжӯЈеёёиҝҗиЎҢ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢиҝҷйҮҢж“ҚдҪңеңЁжҲ‘д»¬жҜҸж¬Ўж–°е»әдёҖдёӘйңҖиҰҒз”ЁеҲ°jpaзҡ„йЎ№зӣ®зҡ„ж—¶еҖҷйғҪйңҖиҰҒйҮҚеӨҚзҡ„еҒҡдёҖж¬ЎгҖӮд№ҹи®ёдҪ еңЁз¬¬дёҖж¬ЎиҮӘе·ұе»әз«ӢйЎ№зӣ®зҡ„ж—¶еҖҷжҳҜеңЁGoogleдёҠиҮӘе·ұжҗңзҙўдәҶдёҖз•ӘпјҢиҠұдәҶеҚҠеӨ©ж—¶й—ҙи§ЈеҶіжҺүдәҶеҗ„з§ҚеҘҮжҖӘзҡ„й—®йўҳд№ӢеҗҺпјҢjpaз»ҲдәҺиғҪжӯЈеёёиҝҗиЎҢдәҶгҖӮжңүдәӣжңүз»ҸйӘҢзҡ„дәәдјҡеңЁOneNoteдёҠйқўжҠҠиҝҷж¬Ўе»әз«ӢйЎ№зӣ®зҡ„иҝҮзЁӢз»ҷи®°еҪ•дёӢжқҘпјҢеҢ…жӢ¬ж“ҚдҪңзҡ„жӯҘйӘӨд»ҘеҸҠйңҖиҰҒз”ЁеҲ°зҡ„й…ҚзҪ®ж–Үд»¶зҡ„еҶ…е®№пјҢеңЁдёӢдёҖж¬ЎеҶҚеҲӣе»әjpaйЎ№зӣ®зҡ„ж—¶еҖҷпјҢе°ұдёҚйңҖиҰҒеҶҚж¬ЎеҺ»GoogleдәҶпјҢеҸӘйңҖиҰҒз…§зқҖз¬”и®°жқҘпјҢд№ӢеҗҺеҶҚжҠҠжүҖжңүзҡ„й…ҚзҪ®ж–Үд»¶copy&pasteе°ұеҸҜд»ҘдәҶгҖӮ

еғҸдёҠйқўиҝҷж ·зҡ„ж“ҚдҪңд№ҹдёҚз®—дёҚиЎҢпјҢдәӢе®һдёҠжҲ‘д»¬еңЁжІЎжңүstarterд№ӢеүҚйғҪжҳҜиҝҷд№Ҳе№Ізҡ„пјҢдҪҶжҳҜиҝҷж ·еҒҡжңүеҮ дёӘй—®йўҳпјҡ

  1. еҰӮжһңиҝҮзЁӢжҜ”иҫғз№ҒзҗҗпјҢиҝҷж ·дёҖжӯҘжӯҘж“ҚдҪңдјҡеўһеҠ еҮәй”ҷзҡ„еҸҜиғҪжҖ§
  2. дёҚеҒңең°copy&pasteдёҚз¬ҰеҗҲDonвҖҷt repeat yourselfзІҫзҘһ
  3. еңЁз¬¬дёҖж¬Ўй…ҚзҪ®зҡ„ж—¶еҖҷпјҲе°Өе…¶еҰӮжһңејҖеҸ‘иҖ…жҜ”иҫғе°ҸзҷҪпјүпјҢйңҖиҰҒиҠұиҙ№жҺүеӨ§йҮҸзҡ„ж—¶й—ҙ

дҪҝз”ЁSpring Boot StarterжҸҗеҚҮж•ҲзҺҮ

starterзҡ„дё»иҰҒзӣ®зҡ„е°ұжҳҜдёәдәҶи§ЈеҶідёҠйқўзҡ„иҝҷдәӣй—®йўҳгҖӮ

starterзҡ„зҗҶеҝөпјҡstarterдјҡжҠҠжүҖжңүз”ЁеҲ°зҡ„дҫқиө–йғҪз»ҷеҢ…еҗ«иҝӣжқҘпјҢйҒҝе…ҚдәҶејҖеҸ‘иҖ…иҮӘе·ұеҺ»еј•е…Ҙдҫқиө–жүҖеёҰжқҘзҡ„йә»зғҰгҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜдёҚеҗҢзҡ„starterжҳҜдёәдәҶи§ЈеҶідёҚеҗҢзҡ„дҫқиө–пјҢжүҖд»Ҙе®ғд»¬еҶ…йғЁзҡ„е®һзҺ°еҸҜиғҪдјҡжңүеҫҲеӨ§зҡ„е·®ејӮпјҢдҫӢеҰӮjpaзҡ„starterе’ҢRedisзҡ„starterеҸҜиғҪе®һзҺ°е°ұдёҚдёҖж ·пјҢиҝҷжҳҜеӣ дёәstarterзҡ„жң¬иҙЁеңЁдәҺsynthesizeпјҢиҝҷжҳҜдёҖеұӮеңЁйҖ»иҫ‘еұӮйқўзҡ„жҠҪиұЎпјҢд№ҹи®ёиҝҷз§ҚзҗҶеҝөжңүзӮ№зұ»дјјдәҺDockerпјҢеӣ дёәе®ғд»¬йғҪжҳҜеңЁеҒҡдёҖдёӘвҖңеҢ…иЈ…вҖқзҡ„ж“ҚдҪңпјҢеҰӮжһңдҪ зҹҘйҒ“DockerжҳҜдёәдәҶи§ЈеҶід»Җд№Ҳй—®йўҳзҡ„пјҢд№ҹи®ёдҪ еҸҜд»Ҙз”ЁDockerе’ҢstarterеҒҡдёҖдёӘзұ»жҜ”гҖӮ

starterзҡ„е®һзҺ°пјҡиҷҪз„¶дёҚеҗҢзҡ„starterе®һзҺ°иө·жқҘеҗ„жңүе·®ејӮпјҢдҪҶжҳҜд»–д»¬еҹәжң¬дёҠйғҪдјҡдҪҝз”ЁеҲ°дёӨдёӘзӣёеҗҢзҡ„еҶ…е®№пјҡConfigurationPropertiesе’ҢAutoConfigurationгҖӮеӣ дёәSpring BootеқҡдҝЎвҖңзәҰе®ҡеӨ§дәҺй…ҚзҪ®вҖқиҝҷдёҖзҗҶеҝөпјҢжүҖд»ҘжҲ‘д»¬дҪҝз”ЁConfigurationPropertiesжқҘдҝқеӯҳжҲ‘д»¬зҡ„й…ҚзҪ®пјҢе№¶дё”иҝҷдәӣй…ҚзҪ®йғҪеҸҜд»ҘжңүдёҖдёӘй»ҳи®ӨеҖјпјҢеҚіеңЁжҲ‘д»¬жІЎжңүдё»еҠЁиҰҶеҶҷеҺҹе§Ӣй…ҚзҪ®зҡ„жғ…еҶөдёӢпјҢй»ҳи®ӨеҖје°ұдјҡз”ҹж•ҲпјҢиҝҷеңЁеҫҲеӨҡжғ…еҶөдёӢжҳҜйқһеёёжңүз”Ёзҡ„гҖӮйҷӨжӯӨд№ӢеӨ–пјҢstarterзҡ„ConfigurationPropertiesиҝҳдҪҝеҫ—жүҖжңүзҡ„й…ҚзҪ®еұһжҖ§иў«иҒҡйӣҶеҲ°дёҖдёӘж–Үд»¶дёӯпјҲдёҖиҲ¬еңЁresourcesзӣ®еҪ•дёӢзҡ„application.propertiesпјүпјҢиҝҷж ·жҲ‘д»¬е°ұе‘ҠеҲ«дәҶSpringйЎ№зӣ®дёӯXMLең°зӢұгҖӮ

starterзҡ„ж•ҙдҪ“йҖ»иҫ‘пјҡ

дёҠйқўзҡ„starterдҫқиө–зҡ„jarе’ҢжҲ‘д»¬иҮӘе·ұжүӢеҠЁй…ҚзҪ®зҡ„ж—¶еҖҷдҫқиө–зҡ„jarе№¶жІЎжңүд»Җд№ҲдёҚеҗҢпјҢжүҖд»ҘжҲ‘д»¬еҸҜд»Ҙи®Өдёәstarterе…¶е®һжҳҜжҠҠиҝҷдёҖдәӣз№Ғзҗҗзҡ„й…ҚзҪ®ж“ҚдҪңдәӨз»ҷдәҶиҮӘе·ұпјҢиҖҢжҠҠз®ҖеҚ•дәӨз»ҷдәҶз”ЁжҲ·гҖӮйҷӨдәҶеё®еҠ©з”ЁжҲ·еҺ»йҷӨдәҶз№Ғзҗҗзҡ„жһ„е»әж“ҚдҪңпјҢеңЁвҖңзәҰе®ҡеӨ§дәҺй…ҚзҪ®вҖқзҡ„зҗҶеҝөдёӢпјҢConfigurationPropertiesиҝҳеё®еҠ©з”ЁжҲ·еҮҸе°‘дәҶж— и°“зҡ„й…ҚзҪ®ж“ҚдҪңгҖӮе№¶дё”еӣ дёә application.properties ж–Үд»¶зҡ„еӯҳеңЁпјҢеҚідҪҝйңҖиҰҒиҮӘе®ҡд№үй…ҚзҪ®пјҢжүҖжңүзҡ„й…ҚзҪ®д№ҹеҸӘйңҖиҰҒеңЁдёҖдёӘж–Үд»¶дёӯиҝӣиЎҢпјҢдҪҝз”Ёиө·жқҘйқһеёёж–№дҫҝгҖӮ

дәҶи§ЈдәҶstarterе…¶е®һе°ұжҳҜеё®еҠ©з”ЁжҲ·з®ҖеҢ–дәҶй…ҚзҪ®зҡ„ж“ҚдҪңд№ӢеҗҺпјҢиҰҒзҗҶи§Јstarterе’Ңиў«й…ҚзҪ®дәҶstarterзҡ„з»„д»¶д№Ӣй—ҙе№¶дёҚжҳҜз«һдәүе…ізі»пјҢиҖҢжҳҜиҫ…еҠ©е…ізі»пјҢеҚіжҲ‘д»¬еҸҜд»Ҙз»ҷдёҖдёӘз»„д»¶еҲӣе»әдёҖдёӘstarterжқҘи®©жңҖз»Ҳз”ЁжҲ·еңЁдҪҝз”ЁиҝҷдёӘз»„д»¶зҡ„ж—¶еҖҷжӣҙеҠ зҡ„з®ҖеҚ•ж–№дҫҝгҖӮеҹәдәҺиҝҷз§ҚзҗҶеҝөпјҢжҲ‘д»¬еҸҜд»Ҙз»ҷд»»ж„ҸдёҖдёӘзҺ°жңүзҡ„з»„д»¶еҲӣе»әдёҖдёӘstarterжқҘи®©еҲ«дәәеңЁдҪҝз”ЁиҝҷдёӘз»„д»¶зҡ„ж—¶еҖҷжӣҙеҠ зҡ„з®ҖеҚ•ж–№дҫҝпјҢдәӢе®һдёҠSpring Bootеӣўйҳҹе·Із»Ҹеё®еҠ©зҺ°жңүеӨ§йғЁеҲҶзҡ„жөҒиЎҢзҡ„з»„д»¶еҲӣе»әеҘҪдәҶе®ғд»¬зҡ„starterпјҢдҪ еҸҜд»ҘеңЁиҝҷйҮҢжҹҘзңӢиҝҷдәӣstarterзҡ„еҲ—иЎЁгҖӮ

еҲӣе»әиҮӘе·ұзҡ„Spring Boot Starter

еҰӮжһңдҪ жғіиҰҒиҮӘе·ұеҲӣе»әдёҖдёӘstarterпјҢйӮЈд№Ҳеҹәжң¬дёҠеҢ…еҗ«д»ҘдёӢеҮ жӯҘ
0. еҲӣе»әдёҖдёӘstarterйЎ№зӣ®пјҢе…ідәҺйЎ№зӣ®зҡ„е‘ҪеҗҚдҪ еҸҜд»ҘеҸӮиҖғиҝҷйҮҢ

  1. еҲӣе»әдёҖдёӘConfigurationPropertiesз”ЁдәҺдҝқеӯҳдҪ зҡ„й…ҚзҪ®дҝЎжҒҜпјҲеҰӮжһңдҪ зҡ„йЎ№зӣ®дёҚдҪҝз”Ёй…ҚзҪ®дҝЎжҒҜеҲҷеҸҜд»Ҙи·іиҝҮиҝҷдёҖжӯҘпјҢдёҚиҝҮиҝҷз§Қжғ…еҶөйқһеёёе°‘и§Ғпјү
  2. еҲӣе»әдёҖдёӘAutoConfigurationпјҢеј•з”Ёе®ҡд№үеҘҪзҡ„й…ҚзҪ®дҝЎжҒҜпјӣеңЁAutoConfigurationдёӯе®һзҺ°жүҖжңүstarterеә”иҜҘе®ҢжҲҗзҡ„ж“ҚдҪңпјҢе№¶дё”жҠҠиҝҷдёӘзұ»еҠ е…Ҙspring.factoriesй…ҚзҪ®ж–Үд»¶дёӯиҝӣиЎҢеЈ°жҳҺ
  3. жү“еҢ…йЎ№зӣ®пјҢд№ӢеҗҺеңЁдёҖдёӘSpringBootйЎ№зӣ®дёӯеј•е…ҘиҜҘйЎ№зӣ®дҫқиө–пјҢз„¶еҗҺе°ұеҸҜд»ҘдҪҝз”ЁиҜҘstarterдәҶ

жҲ‘д»¬жқҘзңӢдёҖдёӘдҫӢеӯҗпјҲдҫӢеӯҗзҡ„е®Ңж•ҙд»Јз ҒдҪҚдәҺhttps://github.com/RitterHou/learn-spring-boot-starterпјү

йҰ–е…Ҳж–°е»әдёҖдёӘMavenйЎ№зӣ®пјҢи®ҫзҪ® pom.xml ж–Үд»¶еҰӮдёӢпјҡ

еҲӣе»әprotertiesзұ»жқҘдҝқеӯҳй…ҚзҪ®дҝЎжҒҜпјҡ

дёҠйқўиҝҷдёӘзұ»е°ұжҳҜе®ҡд№үдәҶдёҖдёӘеұһжҖ§пјҢе…¶й»ҳи®ӨеҖјжҳҜ http://www.baidu.com/пјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮеңЁ application.properties дёӯж·»еҠ й…ҚзҪ® http.url=https://www.zhihu.com жқҘиҰҶзӣ–еҸӮж•°зҡ„еҖјгҖӮ

еҲӣе»әдёҡеҠЎзұ»пјҡ

иҝҷдёӘдёҡеҠЎзұ»зҡ„ж“ҚдҪңйқһеёёз®ҖеҚ•пјҢеҸӘеҢ…еҗ«дәҶдёҖдёӘ url еұһжҖ§е’ҢдёҖдёӘ getHtml ж–№жі•пјҢз”ЁдәҺиҺ·еҸ–дёҖдёӘзҪ‘йЎөзҡ„HTMLж•°жҚ®пјҢиҜ»иҖ…зңӢзңӢе°ұжҮӮдәҶгҖӮ

еҲӣе»әAutoConfiguration

еңЁдёҠйқўзҡ„AutoConfigurationдёӯжҲ‘д»¬е®һзҺ°дәҶиҮӘе·ұиҰҒжұӮпјҡеңЁSpringзҡ„дёҠдёӢж–ҮдёӯеҲӣе»әдәҶдёҖдёӘHttpClientзұ»зҡ„beanпјҢе№¶дё”жҲ‘д»¬жҠҠpropertiesдёӯзҡ„дёҖдёӘеҸӮж•°иөӢз»ҷдәҶиҜҘbeanгҖӮ
е…ідәҺ@ConditionalOnMissingBean иҝҷдёӘжіЁи§ЈпјҢе®ғзҡ„ж„ҸжҖқжҳҜеңЁиҜҘbeanдёҚеӯҳеңЁзҡ„жғ…еҶөдёӢжӯӨж–№жі•жүҚдјҡжү§иЎҢпјҢиҝҷдёӘзӣёеҪ“дәҺејҖе…ізҡ„и§’иүІпјҢжӣҙеӨҡе…ідәҺејҖе…ізі»еҲ—зҡ„жіЁи§ЈеҸҜд»ҘеҸӮиҖғиҝҷйҮҢгҖӮ

жңҖеҗҺпјҢжҲ‘д»¬еңЁ resources ж–Үд»¶еӨ№дёӢж–°е»әзӣ®еҪ• META-INFпјҢеңЁзӣ®еҪ•дёӯж–°е»ә spring.factories ж–Үд»¶пјҢе№¶дё”еңЁ spring.factories дёӯй…ҚзҪ®AutoConfigurationпјҡ
