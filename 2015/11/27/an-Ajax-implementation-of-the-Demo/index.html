一个 Ajax 的实现的 Demo

2015-11-27 · 约 4 分钟阅读

在以前，当人们想要提交一个网页表单的时候，必须提交整个页面，这种操作方式不仅浪费了 CPU 带宽等硬件资源，并且也降低了用户体验，Ajax 技术的出现解决了这一问题。

Ajax（Asynchronous Javascript And XML），是一种通过 JavaScript 来发送 HTTP 请求并且接受 HTTP 响应的方式，JavaScript 在通过 DOM 对 HTML 页面的操作方式下可以在不提交整个表单的时候，实现网页页面的局部与服务器通信并且刷新。
