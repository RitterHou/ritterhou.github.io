Intellj Idea 修改默认的 Jsp 页面模板

2015-12-27 · 约 1 分钟阅读

选择 File → Settings… → File and Code Templates → Other → Web → Jsp files → Jsp File.jsp，然后修改 Jsp File.jsp 中的内容就可以了
