轻量级任务队列 RQ 的简单使用

2022-05-19 · 约 12 分钟阅读

RQ (Redis Queue) 是一个轻量级的 Python 任务队列，这里记录一下它的简单使用。

首先安装 RQ（这里使用的 Python 版本是 3.8.0）

pip install rq==1.10.1

随后创建如下的文件

.
├── __init__.py
├── jobs.py
└── run.py

其中__init__.py中通过连接 redis-server 创建了两个 queue：default 和 queue_1

之后我们在jobs.py中定义需要执行的任务

创建好了任务发送队列和任务本身之后，我们就可以启动 worker 了。我们让 worker 监听两个队列：default 和 queue_1

rq worker queue_1 default --with-scheduler

随后我们就可以在run.py中发送任务给 worker 去执行了
